# Currency converter

Go application to convert currencies by hourly updated prizes on console or by API

## Run

```bash
# help:
go run . -h

# To run in console:
go run . -run=console

# To run API
go run . -run=console
```

## Usage

### API

```url
http://localhost:8080/?amount=1000&input=eur&output=usd
```
